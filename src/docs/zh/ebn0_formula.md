### 1. 术语解释（专业含义与单位）
- **Eb/N0（比特能量与噪声功率谱密度比）**  
  Eb：每比特信号能量（Energy per bit，单位：焦耳/比特）  
  N0：噪声功率谱密度（Noise power spectral density，单位：瓦/赫兹）  
  Eb/N0 是通信系统中衡量信号质量的核心指标，尤其在数字调制（如QPSK、16QAM）中决定误码率（BER）。通常以 **dB** 为单位表示（Eb/N0 (dB)）。

- **C/N（载波功率与噪声功率比）**  
  C：载波功率（Carrier power，单位：瓦）  
  N：在接收机噪声带宽内的总噪声功率（Noise power）  
  C/N 是射频/卫星通信中常见的链路质量指标，反映接收端信号强度与噪声的对比。同样以 **dB** 为单位。

- **Rb（比特率 / Bit Rate）**  
  单位：kbps（千比特/秒）。表示每秒传输的比特数。本例中 500 kbps = 500,000 bps。

- **Bn（噪声带宽 / Noise Bandwidth）**  
  单位：kHz（千赫兹）。指接收机滤波器中噪声所占据的有效带宽（通常等于或略大于信号带宽）。本例中 333 kHz = 333,000 Hz。

**关键点**：Eb/N0 和 C/N **不是同一个量**，它们之间通过“比特率与噪声带宽的比值”转换。这就是该转换器存在的意义。

### 2. 核心公式与严格推导
**线性形式**：
\[
\frac{C}{N} = \left( \frac{E_b}{N_0} \right) \times \left( \frac{R_b}{B_n} \right)
\]

**推导过程**（物理意义）：
1. 载波功率 C = Eb × Rb  
   （每比特能量 × 每秒比特数 = 功率，单位：J/bit × bit/s = W）

2. 噪声功率 N = N0 × Bn  
   （噪声谱密度 × 带宽 = 总噪声功率，单位：W/Hz × Hz = W）

3. 两式相除 → C/N = (Eb/N0) × (Rb/Bn)

**dB 形式**（最常用）：
\[
\left( \frac{C}{N} \right)_{\rm dB} = \left( \frac{E_b}{N_0} \right)_{\rm dB} + 10 \log_{10} \left( \frac{R_b}{B_n} \right)
\]

反向转换：
\[
\left( \frac{E_b}{N_0} \right)_{\rm dB} = \left( \frac{C}{N} \right)_{\rm dB} - 10 \log_{10} \left( \frac{R_b}{B_n} \right)
\]

**注意**：Rb 必须用 bps，Bn 必须用 Hz（工具自动处理 kbps/kHz 单位转换，比值不变）。

### 3. 工具界面数值验证（精确计算）
工具中 Rb = 500 kbps，Bn = 333 kHz →  
\[
\frac{R_b}{B_n} = \frac{500\,000}{333\,000} = \frac{500}{333} \approx 1.5015015015
\]
\[
10 \log_{10}(1.5015015015) \approx 1.765\,{\rm dB}
\]

### 4. 原理说明（为什么需要这个转换器？）
- 在基带数字通信（如 LTE、5G、卫星链路）中，误码率曲线通常用 **Eb/N0** 画出（因为它直接与调制方式相关）。
- 但实际链路预算（Link Budget）中，射频前端测量的是 **C/N**（因为测量仪表显示的是载波功率与噪声功率）。
- 因此必须在“每比特能量”和“总载波功率”之间转换，而转换因子正是 **Rb/Bn**（即“每个比特占用多少带宽”）。
- 当 Rb > Bn（本例 500 > 333），C/N 会比 Eb/N0 大（正的 dB 修正）；反之则变小。这完全符合信息论和信号处理原理。
